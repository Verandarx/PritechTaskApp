# Task Manager App (React Native + Expo)

This is a simple Task Manager mobile application built with React Native and Expo.  
The purpose of this project is to practice mobile development concepts such as state management, navigation, and local storage.

---

## Project Overview

The app allows users to create, manage, and organize daily tasks in a simple and efficient way.  
It focuses on clean structure, usability, and basic mobile app functionality.

---

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Search tasks
- Save tasks locally using AsyncStorage
- Simple navigation between screens

---

## Tech Stack

- React Native
- Expo
- JavaScript / TypeScript
- React Navigation
- AsyncStorage
