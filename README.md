# Task Manager App (React Native + Expo)

This is a simple Task Manager mobile application built with React Native and Expo. The purpose of this project is to practice mobile development concepts such as state management, navigation, local storage, and working with external APIs.

## Project Overview

The app allows users to create, manage, and organize daily tasks in a simple and efficient way. It focuses on clean structure, usability, and core mobile app functionality.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Search tasks
- Save tasks locally using AsyncStorage
- Simple navigation between screens

## Tech Stack

- React Native
- Expo
- JavaScript / TypeScript
- React Navigation
- AsyncStorage

## Project Structure

The project uses Expo Router file-based structure:

app/
  _layout.tsx
  index.tsx
  explore.tsx

components/
assets/

## Installation & Setup

1. Clone the repository  
git clone https://github.com/Verandarx/PritechTaskApp.git

2. Navigate into the project  
cd PritechTaskApp

3. Install dependencies  
npm install

4. Start the project  
npx expo start

Then scan the QR code using Expo Go or run it in an emulator/browser.

## API Integration

This app fetches data from a public API to demonstrate working with external data sources and handling dynamic content in a React Native application.
